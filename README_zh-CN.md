# deepin-riscv

这是一个将deepin操作系统移植到RISC-V平台的项目，如果有项目需要进行修改我们会在此进行归档，后续会尽量合入主线分支

## 目前适配的硬件平台
基于StarFive JH7100 SoC [VisionFive](https://github.com/starfive-tech/VisionFive/blob/main/README.md) 

## 当前目标
基于debian-ports的环境去适配dde的桌面环境
